# Dataset notes

`Car_details_v3.csv` is the CarDekho used-car dataset used in this project. The original source is available on [Kaggle](https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho).

It contains historical vehicle listings. `selling_price` is in Indian Rupees (INR). A few specification columns intentionally contain missing values or units embedded in text; the notebook cleans these transparently rather than silently discarding the issue.
